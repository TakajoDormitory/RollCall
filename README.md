# 点呼チェックシステム

</div>
高城寮の点呼チェックシステムです。  
宿直教員がiPadで点呼を行えるように、また週番が点呼チェックをしなくてよくなるように開発しています。

## バグ、問題発見
[テンプレート](https://github.com/TakajoDormitory/RollCall/issues/new/choose)にしたがって、[issue](https://github.com/TakajoDormitory/HomePage/issues)の投稿よろしくお願いします。
